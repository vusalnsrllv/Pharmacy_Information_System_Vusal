# Онлайн-информационная справочная система для аптек «АптекаПлюс»

Данный проект представляет собой современный веб-интерфейс информационной системы, разработанный в рамках курсовой работы. Система предназначена для удобного поиска лекарственных средств, ознакомления с их характеристиками и поиска ближайших аптечных пунктов.

## 🚀 Функциональные возможности
* **Интерактивный каталог:** Фильтрация медикаментов по категориям (Обезболивающие, Витамины, Антибиотики и др.).
* **Умный поиск:** Поиск препаратов в реальном времени по названию.
* **Карточки товаров:** Детальная информация о каждом препарате (описание, дозировка, противопоказания) в модальных окнах.
* **Интерактивная карта:** Визуализация расположения аптек с отображением статуса (открыто/закрыто) и ассортимента.
* **Раздел FAQ:** Ответы на часто задаваемые вопросы пользователей.
* **Форма связи:** Реализованная валидация и имитация отправки сообщений.

## 🛠 Технологический стек
* **HTML5:** Семантическая разметка для обеспечения доступности и SEO.
* **CSS3:** Современная верстка с использованием CSS-переменных, Flexbox и CSS Grid. Дизайн выполнен в сдержанном медицинском стиле с использованием шрифтов *Playfair Display* и *DM Sans*.
* **JavaScript (ES6+):** Реализация всей логики приложения без использования сторонних библиотек (Vanilla JS). Управление состоянием страниц (SPA-подход), динамический рендеринг данных и фильтрация.
